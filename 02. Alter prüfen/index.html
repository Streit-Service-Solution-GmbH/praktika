Das Programm soll überprüfen ob ein Benutzer volljährig ist.

Der Benutzer soll sein Alter in ein Eingabefeld eingeben.

Über einen Button soll dann eine Funktion aufgerufen werden, welche prüft ob das eingegebene Alter über 18 Jahren liegt.

Je nachdem ob der Benutzer volljährig ist oder nicht, soll eine entsprechende Meldung auf der Oberfläche ausgegeben werden.
