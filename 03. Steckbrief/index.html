Das Programm soll einen Steckbrief für einen Benutzer generieren.

Der Benutzer soll die Möglichkeit haben, seinen Namen, Nachnamen, Anschrift, Geburtsdatum, Geburtsort und sein Lieblingsessen in mehrere Eingabefelder einzutragen.

Über einen Button soll dann ein Steckbrief erzeugt werden, der die eingegebenen Daten nutzt.

Der Steckbrief soll sich in der Mitte des Bildschirms befinden und mittels CSS anschaulich gestaltet werden.
